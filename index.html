I'm a fourth-year undergraduate student in Computer Science at Stanford University, advised by Keith Winstein.

My interests lie at the intersection of computer systems, formal methods, and software engineering.

I want safe and correct systems to become the norm. To further that vision, I mostly work on tools that make writing verified code more ergonomic and tools that make secure code more efficient. I am also thinking about how LLMs can help humans write correct code and how undergraduate students should be learning about verification. Please reach out if you want to discuss!
